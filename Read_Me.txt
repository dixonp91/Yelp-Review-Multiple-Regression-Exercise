Yelp Rating Regression Predictor
challenge project on Codecademy for learning Regression

Project uses data from Yelp to evaluate factors that contribute to a high yelp rating using multiple Regression
Steps:
    - Use Jupyter Notebook to organize project
    - import data and evaluate, Pandas
    - merge and clean data, Pandas
    - evaluate data frame for correlations, Matplotlib
    - split data into training and validation sets, sklearn
    - create model, sklearn
